## 🔑 关键要点总结

### 提醒设置技巧：

1. **精准定位**：设置合适的形态参数避免误报
2. **分级提醒**：根据重要性设置不同通知方式
3. **确认机制**：添加确认条件提高准确性
4. **定期维护**：根据市场变化调整提醒参数

### 常见提醒问题：

1. **提醒过多**：条件过于宽松导致频繁打扰
2. **提醒延迟**：网络或系统原因导致通知延迟
3. **误报/漏报**：参数设置不当导致识别问题
4. **重复提醒**：相同条件触发多次提醒
